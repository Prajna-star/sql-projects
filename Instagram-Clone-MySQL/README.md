## SQL Concepts Demonstrated

This project demonstrates practical SQL skills using a simplified Instagram database.

### Concepts Covered

- Database Design
- Primary & Foreign Keys
- One-to-Many Relationships
- Many-to-Many Relationships
- INNER JOIN
- LEFT JOIN
- Aggregate Functions
- GROUP BY
- HAVING
- Subqueries
- Sorting & Filtering
- Business Analytics Queries

### Business Questions Answered

- Who were the first users to join the platform?
- Which weekday attracts the highest number of new registrations?
- Which users have never uploaded a photo?
- Which photo received the most likes?
- What is the average number of photos uploaded per user?
- What are the platform's most popular hashtags?
- Which users have liked every photo on the platform?
